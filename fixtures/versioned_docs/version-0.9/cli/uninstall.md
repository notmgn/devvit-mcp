## `devvit uninstall APPSLUG SUBREDDIT`

Uninstall an app

```
USAGE
  $ devvit uninstall APPSLUG SUBREDDIT

ARGUMENTS
  APPSLUG    The slug of the app you want to uninstall
  SUBREDDIT  Uninstall from the given subreddit. The "r/" prefix is optional

DESCRIPTION
  Uninstall an app
```
