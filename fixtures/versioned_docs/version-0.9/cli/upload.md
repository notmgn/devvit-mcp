## `devvit upload`

Upload the app to the App Directory

```
USAGE
  $ devvit upload [-b major|minor|patch|prerelease]

FLAGS
  -b, --bump=<option>  Type of version bump (major|minor|patch|prerelease)
                       <options: major|minor|patch|prerelease>

DESCRIPTION
  Upload the app to the App Directory
```
