## `devvit bundle actor NAME`

Bundle an actor into bundle.json

```
USAGE
  $ devvit bundle actor [NAME]

ARGUMENTS
  NAME  The name of of the actor to bundle

DESCRIPTION
  Bundle an actor into bundle.json

EXAMPLES
  $ devvit bundle actor my-actor
```
